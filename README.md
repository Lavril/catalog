# Cправочник Организаций, Зданий, Деятельности
Асинхронный REST API справочника. 

## Организации
Представляет собой карточку организации и содержит следующую информацию:

- Название
- Номер телефона (может иметь несколько)
- Здание
- Деятельность (может иметь несколько видов)

## Здания
Информация о конкретном здании

- Адрес
- Широта
- Долгота

## Деятельность
Позволяет классифицировать род деятельности организаций в справочнике. Имеет название и может в древовидном виде вкладываться друг в друга. Уровень вложенности деятельностей - 3 уровня.

## Стек
- FastAPI
- SQLAlchemy 2.0 (async)
- PostgreSQL
- Alembic
- Docker
# Быстрый старт
## Руководство по быстрому запуску приложения.

1. Установка зависимостей
```bash
make install
```

2. Необходимо создать файл `.env` с переменными окружения на основе файла `.env.template`.
3. Запуск контейнеров
```bash
make up
```
Будут подняты:

- db — PostgreSQL
- api — FastAPI приложение

4. Создание миграции
```bash
make init
```

5. Применение миграции
```bash
make migrate
```

6. Заполнение базы данных тестовыми данными
```bash
make seed
```
API будет доступен по ссылке:

http://localhost:8000

## Swagger
http://localhost:8000/openapi

API ключ из переменных окружения.

## Удаление контейнеров и БД
```bash
make down
```

# Функционал

- Список всех организаций находящихся в конкретном здании (`/organizations/by-building/{building_id}`)
- Список всех организаций, которые относятся к указанному виду деятельности (`/organizations/by-activity/{activity_id}`)
- Список всех организаций, которые относятся к указанному виду деятельности (включая вложенные) (`/organizations/by-activity-tree/{activity_id}`)
- Список всех зданий (`/buildings/`)
- Список всех зданий в прямоугольной области (`/buildings/in-area`)
- Список всех деятельностей (`/activities/`)
- Список организаций, которые находятся в заданном радиусе/прямоугольной области относительно указанной точки на карте. (`/organizations/in-raduis` / `/organizations/in-rectangle`)
- Вывод информации об организации по её идентификатору (`/organizations/{org_id}`)
- Поиск организации по названию (`/organizations/search`)
